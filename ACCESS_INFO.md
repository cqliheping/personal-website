## 🖥️ 如何访问

### 方法 1: 直接在浏览器打开
```
http://10.15.10.32:8000
```

### 方法 2: 如果无法访问
1. **检查网络连接**
   ```bash
   ping 10.15.10.32
   ```

2. **检查防火墙**
   ```bash
   # 如果被阻止，开放端口 8000
   sudo ufw allow 8000/tcp
   ```

3. **确认端口开放**
   ```bash
   telnet 10.15.10.32 8000
   # 或
   nc -zv 10.15.10.32 8000
   ```

## 🔄 服务器管理

### 查看服务器状态
```bash
ps aux | grep "python3 -m http.server"
```

### 停止服务器
```bash
ps aux | grep "python3 -m http.server" | grep -v grep | awk '{print $2}' | xargs kill
```

### 重启服务器
```bash
cd /mnt/data2/test
python3 -m http.server 8000 --bind 0.0.0.0 &
```

### 更改端口（如果 8000 被占用）
```bash
cd /mnt/data2/test
python3 -m http.server 8080 --bind 0.0.0.0 &
# 访问 http://10.15.10.32:8080
```
